## Primeiros passos

Primeiro, instale as dependências:

```
Instalação do npm
```

Em segundo lugar, execute o servidor de desenvolvimento:

```
npm executar dev
```

Abra [http://localhost:3000](http://localhost:3000) com seu navegador para ver o resultado.
